#DragPan

A jQuery plugin designed to allow click and drag panning of an element.


##Usage

    <div id="document">
      <div id="content">
      </div>
    </div>
    <script src="path/to/jquery.js"></script>
    <script src="path/to/dragpan.js"></script>
    <script>
    $(document).ready( function () {
      $("#document").dragpan();
    });
    </script>
